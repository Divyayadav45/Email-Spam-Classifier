# SMS Spam Classifier

This project is a Machine Learning based SMS Spam Detection system built using Python and Scikit-Learn. The model predicts whether a given SMS message is **Spam** or **Not Spam (Ham)**.

## 🚀 Features
- Detects spam SMS messages
- Built using Natural Language Processing (NLP)
- Uses TF-IDF vectorization
- Trained using a Naive Bayes classifier
- Simple interface for prediction

## 🧠 Machine Learning Workflow
1. Data Cleaning
2. Text Preprocessing
3. Feature Extraction using TF-IDF Vectorizer
4. Model Training using Naive Bayes
5. Model Serialization using Pickle
6. Prediction through Python app
