## 🧠 About the Project
This project is a **Machine Learning-based SMS Spam Detection System** that classifies messages as **Spam** or **Ham (Not Spam)**.  

It leverages **Natural Language Processing (NLP)** techniques along with **TF-IDF Vectorization** and a **Naive Bayes classifier** to deliver accurate predictions.

---

## ✨ Features
- 📩 Detects spam SMS messages efficiently  
- 🧠 NLP-based text processing  
- 🔍 TF-IDF feature extraction  
- ⚡ Fast and lightweight Naive Bayes model  
- 💻 Simple Python-based prediction interface  

---

## 🔄 ML Workflow
- 🧹 Data Cleaning  
- 🔤 Text Preprocessing (Tokenization, Stopword Removal)  
- 📊 Feature Extraction using TF-IDF  
- 🤖 Model Training using Naive Bayes  
- 💾 Model Saving using Pickle  
- 🔮 Prediction using Python App
